Robot Framework örnek proje

Hızlı başlangıç:

1. Sanal ortam oluşturun:
   python3 -m venv .venv
2. Sanal ortamı etkinleştirin:
   source .venv/bin/activate
3. Paketleri yükleyin:
   pip install -r requirements.txt
4. Örnek testi çalıştırın:
   robot tests/example.robot

Not: VS Code'da Python uzantısını ve Robot Framework LSP uzantısını yüklemeniz önerilir.
